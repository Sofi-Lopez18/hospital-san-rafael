# Proyecto de Gestión de pasantes Hospital San Rafael 

## Descripción del proyecto

Es una aplicación web desarrollada con Spring Boot 3.2.5, Java 21, PostgreSQL y Thymeleaf. Su objetivo es gestionar el ciclo completo de los estudiantes en práctica clínica: registro, asignación de horarios y turnos, control de ingreso/salida (check in/out), seguimiento de documentación y perfil académico. Tiene dos tipos de usuarios principales: el personal administrativo (administrador, director, médico, docente) y los estudiantes.

---

# Tecnologías utilizadas

- Java 21
- Spring Boot 3.2.5
- Spring Security + BCrypt
- PostgreSQL
- Hibernate / Spring Data JPA
- HikariCP
- Thymeleaf
- Apache Tomcat (embebido)
- Apache Maven
- HTML5, CSS3, JavaScript Vanilla, SVG
